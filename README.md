# MISHARP NEWS POST

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 포함 기능
- 서울 실시간 날씨 / 공기지수
- 패션·유통·IT·온라인마케팅 뉴스
- 주요 경제뉴스
- 기사 더보기 / 접기
- 관련 정보 사이트 20개
- 버튼 클릭형 미샵 인사이트
- 개인정보처리방침 · 서비스약관 페이지

## 선택 설정
Streamlit secrets에 아래를 넣으면 더 좋습니다.
- OPENAI_API_KEY
- SITE_URL
- OG_IMAGE_URL
